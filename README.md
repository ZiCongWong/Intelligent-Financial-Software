###### 开发前的配置要求

1. 安装Node.js
2. 安装HBuilder

###### **安装步骤**

1. Clone the repo
2. 在HBuilder打开

```sh
git clone https://github.com/ZiCongWong/Intelligent-Financial-Software.git
```
